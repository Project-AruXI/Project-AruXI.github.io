# Project AruXI — Documentation Site

This repository contains the Jekyll site for Project AruXI — a new computer architecture, assembly, and language. The site holds documentation, design notes, and tutorials.

Local development

1. Install dependencies (Ruby + Bundler). Then:

```bash
bundle install
bundle exec jekyll serve --livereload
```

Open http://localhost:4000 to view the site.

Contributing

Add docs under the `docs/` directory. Pages use the `default` layout.
